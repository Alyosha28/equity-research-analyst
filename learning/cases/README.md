# Case Memory

Store compact case summaries here after a completed report, critique, or refresh.

Each case should include:

- Company and ticker.
- Task mode: Mode A report, Mode B critique, or Mode C refresh.
- Key source tiers used.
- Most important valuation assumptions.
- Gate failures and route-back decisions.
- User corrections.
- Reusable lesson, if any.

Do not store raw transcripts or proprietary reports.
