# Learning Artifacts

This directory stores curated, reviewable learning artifacts for the
`equity-research-analyst` skill family.

It is not a raw transcript store. Do not place conversation logs, proprietary
reports, credentials, or unredacted paid research here.

Allowed artifacts:

- `cases/`: short case summaries from completed research tasks.
- `source-cards/`: source quality notes and report ingestion cards.
- `proposals/`: proposed high-risk updates awaiting human review.
- `regression-results/`: replay summaries from eval and smoke runs.

High-risk updates to `SKILL.md`, methodology references, rubrics, report
templates, or valuation engine code require human review and regression replay
before they enter active runtime.
